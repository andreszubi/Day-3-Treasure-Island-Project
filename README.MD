# Treasure Island Project

## Overview
A text-based adventure game written in Python where players navigate through a treasure hunt by making choices at different decision points.

## Project Structure
- `Treasure-Island-Project.py` - Main game file containing the interactive story

## How to Run
Execute the Python file to start the game:
```bash
python Treasure-Island-Project.py
```

## Game Description
Players start at a crossroad and must make strategic choices to find the treasure:
- Choose between going left or right
- Decide whether to wait for a boat or swim across a lake
- Select one of three colored doors (red, yellow, or blue)

Only the correct combination of choices leads to finding the treasure!
